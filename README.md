# honestyFood

**TO RUN EXECUTE FOLLOWING COMMAND IN TERMINAL**
```
1) $ git clone https://github.com/freakk101/honestyFood.git
2) $ cd honestyFood && npm install
3) $ cd frontend && npm install
```
```
4) $ npm start
```
**OPEN NEW TERMINAL TAB (cmd + T)**
```
5) $ cd.. npm start
```
